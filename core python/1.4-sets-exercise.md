### 1. Creating Sets
**Important Note:** `{}` creates an empty dictionary, not a set. Use `set()` for an empty set.

```python
# Correct way to create an empty set
empty_set = set()
print("Empty set:", empty_set)

# This is a dictionary, not a set
empty_dict = {}
print(f"Value {{}} type is: {type(empty_dict)}")

# Creating a set with values
numbers = {1, 2, 3, 4, 5}
print("Initial set:", numbers)

# Create an immutable set (frozenset)
immutable_set = frozenset([1, 2, 3])
print("Immutable set:", immutable_set)
```

### 2. Adding and Removing Elements
**Difference:** `remove()` throws an error if the item is missing. `discard()` does not.

```python
s = {1, 2, 3, 4, 5}

# Add element
s.add(6)
print("After add(6):", s)

# Remove specific element (3) without error if not found
s.discard(3)
print("After discard(3):", s)

# Remove specific element with error (if element doesn't exist)
try:
    s.remove(10) # This will raise KeyError because 10 is not there
except KeyError:
    print("Error: Element 10 not found using remove()")

# Remove all elements (Clear the set)
s.clear()
print("After clear():", s)

# Delete the variable completely
del s
```

### 3. Set Mathematical Operations
Includes Union, Intersection, Difference, and removing intersection.

```python
set_a = {1, 2, 3, 4, 5}
set_b = {2, 3, 4, 5, 6}

print("Union:", set_a.union(set_b)) or set_a|set_b (pipe operator)
print("Intersection:", set_a.intersection(set_b))
print("Difference (A - B):", set_a.difference(set_b))
print("Symmetric Difference:", set_a.symmetric_difference(set_b))

# Remove intersection of set_b from set_a (Modifies set_a in place)
set_a.difference_update(set_b)
print("Set A after removing intersection:", set_a)
```

### 4. Statistics and Properties
Includes Max, Min, Length, and sorting words.

```python
num_set = {10, 50, 5, 99, 2}
```

### 5. Conversions (List, Tuple, Set)
Clean conversions between data types.

```python
# Set to Tuple
my_set = {1, 2, 3}
my_tuple = tuple(my_set)
print("Set to Tuple:", my_tuple)

# Tuple to Set
t = (4, 5, 6, 6)
t_to_set = set(t)
print("Tuple to Set:", t_to_set)

# List to Set (Removes duplicates)
raw_list = [1, 2, 3, 4, 5, 6, 6, 7, 7, 8, 8, 8]
unique_list = list(set(raw_list))
print("List with duplicates removed:", unique_list)
```

### 6. Advanced Logic & Algorithms

#### A. Unique Elements in Two Sets & Union of 3
```python
s1 = {1, 2, 3}
s2 = {4, 2, 5}
# Unique elements in both sets combined (Union)
print("Unique elements:", s1.union(s2))

# Union of 3 sets
a = {1, 2}
b = {2, 3}
c = {3, 4}
print("Union of 3 sets:", a.union(b, c))
```

#### B. Common Elements in 3 Lists
```python
l1 = [1, 2, 3, 4]
l2 = [2, 3, 5, 6]
l3 = [3, 4, 2, 7]

# Convert to sets and find intersection
common = set(l1) & set(l2) & set(l3)
print("Common elements in 3 lists:", common)
```

#### C. Check if Two Lists are Disjoint
```python
list_a = [1, 2, 3]
list_b = [4, 5, 6]

# Disjoint means they have NO common elements
is_disjoint = set(list_a).isdisjoint(set(list_b))
print(f"Are lists disjoint? {is_disjoint}")
```

#### D. Elements Not Present in Another List
```python
main_list = [1, 2, 3, 4, 5]
exclude_list = [2, 4]

# Elements in main_list but not in exclude_list
result = set(main_list) - set(exclude_list)
print("Elements not in exclude list:", result)
```

#### E. Find Missing Numbers from a Range
```python
data_list = [1, 2, 3, 5, 6, 9, 10] # Range is 1 to 10
full_range = set(range(1, 11))
current_set = set(data_list)

missing = full_range - current_set
print("Missing numbers:", sorted(missing))
```

#### F. Generate Unique Random Numbers
```python
import random

# Generate 5 unique random numbers between 1 and 100
random_nums = set()
while len(random_nums) < 5:
    random_nums.add(random.randint(1, 100))

print("Unique random numbers:", random_nums)
```
